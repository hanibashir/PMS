
Perjuf_Management_System (PMS) - School System. 
(Perjuf = Perspektiven Für Junge Flüchtlinge).

A simple java desktop application with MySql database.
------------------------------------------------------
Features:

1)Manage users (Add,Delete and Search).
2)Manage staff with photo(Add,Delete and Search).
3)Manage students(Add,Delete and Search).
4)Manage remarks.

------------------------------------------------------

This application needs the following to work:

1)Apache http server and MySql database server - download it from here: 
https://www.apachefriends.org/de/index.html

2)java - download it here: 
http://www.java.com/en/download/windows_xpi.jsp

All others are included.
